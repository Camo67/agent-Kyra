# KYRA Code Workspace

This directory contains temporary code files created by KYRA during development and experimentation.

## Purpose
- Store temporary `.ts`, `.js`, `.py` files created during development
- Allow for quick prototyping and code sharing
- Serve as a scratchpad for code that may be reviewed later

## Organization
- `js/` - JavaScript files
- `ts/` - TypeScript files  
- `py/` - Python files
- `archive/` - Archived code that's no longer needed
- `review/` - Code awaiting review

## Cleanup Policy
Files older than 30 days will be moved to archive automatically via scheduled n8n workflow.

## Permissions
This directory is intended for read/write access by KYRA agents and Cameron for development purposes.
