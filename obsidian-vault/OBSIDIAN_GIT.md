# Obsidian Git Integration

## Using the "Obsidian Git" Plugin

The most popular way to handle repositories is through the Obsidian Git plugin.

### Obsidian Forum

### Cloning a Repo
You can use the command "Obsidian Git: Clone an existing remote repo" from the Command Palette (Ctrl/Cmd + P).

### Pulling Changes
Once a repository is linked, the plugin can automatically pull changes when you start Obsidian or at set intervals.

### Pushing/Committing
It allows you to commit and push your notes back to GitHub or GitLab directly from the Obsidian interface.
